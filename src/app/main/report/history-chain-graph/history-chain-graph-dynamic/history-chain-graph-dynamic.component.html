This gojs-angular sample demonstrates how to use the Diagram, Palette, and Overview gojs-angular components, made and maintained by Northwoods Software.

Notice how when a Node or Link is removed or added to the Diagram, the list of Nodes and Links the top-level App Component updates accordingly.

For more information on how to use Angular with GoJS, see the GoJS with Angular intro page.
